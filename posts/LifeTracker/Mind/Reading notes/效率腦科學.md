PM

- <span style='color:black'>當沒有幹勁的時候對自己說</span>
  - <span style='color:black'>只要我開始行動，幹勁自然就會產生</span>
  - <span style='color:black'>你想要的東西是甚麼?他在人生遊戲中，所以去玩吧</span>
  - <span style='color:black'>只有今天而已!</span>
- <span style='color:black'>語言可以達到心理振奮</span>
  - <span style='color:black'>同樣的事可以說得很正面/負面</span>
    - <span style='color:black'>正面: 清楚的表達、知道下一步</span>
    - <span style='color:black'>負面: 模糊的表達、不重要的資訊、雜音</span>
- <span style='color:black'>讓行動支配感情</span>
  - <span style='color:black'>先動手做</span>
  - <span style='color:black'>設定好一個完全不做其他事的時間</span>
  - <span style='color:black'>慢慢延長這個時間</span>
- <span style='color:black'>自我動機檢測表: 讓動機下降的因素、具體解方</span>
- <span style='color:black'>單純享受日常，在某瞬間群神貫注在一件所逤的是上，就是莫大的意義</span>
- <span style='color:black'>以周為單位的賞罰</span>
- <span style='color:black'>專注: 想像自己在玻璃瓶內</span>
- <span style='color:black'>相信自己是一流選手，是冠軍，是最厲害的</span>
- <span style='color:black'>最重要的事排在周一到三的上午各三小時</span>
- <span style='color:black'>給自己設Deadline rush，某件事一定要一小時內完成，效率才會高，避免Parkinon's effect(所需時間會一直膨脹)</span>
- <span style='color:black'>在相同主題切換不同模式的工作，每一小段工作就休息一小段</span>
- <span style='color:black'>一次做好一件事</span>
- <span style='color:black'>桌子乾淨</span>
- <span style='color:black'>自我圖</span>
  - <span style='color:black'>Parent: How parents’ behavior affect personality</span>
    - <span style='color:black'>Critical: 做甚麼都不好，束縛</span>
      - <span style='color:black'>CP: 責任感強追求理想，強調原則有攻擊性，盤起雙手臭臉</span>
    - <span style='color:black'>Nurturing: 希望孩子能長大，保護</span>
      - <span style='color:black'>NP: 無法對他人施壓，常等待指示，喜歡溫柔待人。很體貼，時會過度體貼</span>
  - <span style='color:black'>Child: 童年心情</span>
    - <span style='color:black'>Free: 自由天真爛漫</span>
      - <span style='color:black'>好奇心強，精力旺盛，自我中心，衝動強。幽默感佳，喜歡比手畫腳</span>
    - <span style='color:black'>Adapted: 學習適應社會規範</span>
      - <span style='color:black'>非常合群，依賴他人，消極膽小，忍耐度高。</span>
  - <span style='color:black'>Adult: 成長經驗塑造的個性</span>
    - <span style='color:black'>很冷靜理性，講道理，常過度冷靜，常有思考動作。</span>
- <span style='color:black'>讚美與責備</span>
  - <span style='color:black'>讚美: 小細節、具體且頻繁、讚美過程非結果</span>
  - <span style='color:black'>責備: 注意場合時機方式</span>
    - <span style='color:black'>記住人不是為了公司而工作，人是為了提升自我，享受過程</span>
    - <span style='color:black'>先自我批評再批評他人</span>
  - <span style='color:black'>CP: 用肯定的方式責備，少責備</span>
  - <span style='color:black'>NP: 責備完要關心，不要否定對方的才華</span>
  - <span style='color:black'>A: 三明治法</span>
  - <span style='color:black'>FC: 多責備，越挫越勇</span>
  - <span style='color:black'>AC: 多鼓勵</span>
- <span style='color:black'>心流</span>
  - <span style='color:black'>想像自己做得很好</span>
  - <span style='color:black'>忽略沉默成本: 我在當下已經做最好了</span>
  - <span style='color:black'>工作十五分鐘會進入專注狀態，之後開始漸弱</span>
