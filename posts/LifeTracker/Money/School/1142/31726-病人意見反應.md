- 立場：和信醫院是非營利公益醫療單位，目標是改變生命。

# <span style='color:#1E4E79'>病人服務品質回饋與改善分析紀錄</span>
<span style='background:white'>日期與時間：\*\*2026-03-17 16:27:13</span>
<span style='font-weight:bold;background: white'>地點：</span><span style='background:white'>和信醫院五南病房</span>
<span style='font-weight:bold;background: white'>對象：</span><span style='background:white'>小名先生家屬</span>

<span style='font-weight:bold;background: white'>一、概覽</span>
<span style='background:white'>本次溝通由本院病人服務中心主動致電一位曾於本院就醫且已轉院之病人／家屬，針對住院期間之環境清潔與照護支援差異進行意見蒐集。</span>
<span style='background:white'>該名病人曾入住「五北」及「五南」病房，並於三月初在病房內發現蟑螂，對「五南」病房清潔品質未達標準及夜班護理人力不足表達不滿，並認為與「武北」相比，照護助理支援存在明顯差距。</span>
<span style='font-weight:bold;background: white'>主要痛點如下：</span>
- <span style='background:white'>病房清潔品質不佳（包含蟑螂出現、未確實拖地、拖把交叉使用）</span>
- <span style='background:white'>夜間護理人力不足，增加家屬與護理人員負擔</span>
- <span style='background:white'>「住院整合照護試辦計畫」僅於武北提供，造成服務不均</span>
<span style='font-weight:bold;background: white'>病人期待：</span>
<span style='background:white'>強化資源配置公平性、提升清潔品質、擴大照護助理服務範圍，以降低感染風險並減輕照護負擔。</span>

<span style='font-weight:bold;background: white'>二、背景說明</span>
- <span style='background:white'>\*\*通話發起單位：\*\*本院病人服務中心（目的：蒐集回饋以改善服務品質）</span>
- <span style='background:white'>\*\*對象身份：\*\*曾入住五北與五南病房之病人／家屬（目前已轉院）</span>
- <span style='background:white'>\*\*關鍵時間：\*\*三月初於病房內發現蟑螂</span>
- <span style='background:white'>\*\*住院情況補充：\*\*填寫問卷時為五南一般病房（非隔離病房）</span>
<span style='font-weight:bold;background: white'>主要議題：</span>
- <span style='background:white'>病房清潔問題（害蟲、未落實清掃拖地、清潔工具交叉使用）</span>
- <span style='background:white'>照護支援差異（武北有整合照護助理，武南無）</span>
- <span style='background:white'>夜間人力不足問題</span>
<span style='font-weight:bold;background: white'>院方回應摘要：</span>
- <span style='background:white'>已通知清潔廠商改善並更換部分清潔工具</span>
- <span style='background:white'>「住院整合照護」為試辦計畫，目前僅於武北實施</span>
- <span style='background:white'>醫護人力整體不足，院方正持續補充與調整</span>

<span style='font-weight:bold;background: white'>三、主要問題（痛點分析）</span>
<span style='font-weight:bold;background: white'>（一）病房清潔品質未達標準，且存在感染風險</span>
<span style='font-weight:bold;background: white'>問題說明：</span>
<span style='background:white'>病人於三月初在病房內發現蟑螂，且清潔人員未落實日常清掃及拖地作業，甚至需由家屬自行清潔。另有病房與廁所使用同一拖把之情形，恐造成交叉污染。</span>
<span style='font-weight:bold;background: white'>影響：</span>
- <span style='background:white'>提高院內感染風險</span>
- <span style='background:white'>增加病人與家屬心理壓力</span>
- <span style='background:white'>提升照護負擔</span>
- <span style='background:white'>影響醫院形象與信任度</span>
<span style='font-weight:bold;background: white'>現況：</span>
<span style='background:white'>院方雖已要求改善並更換工具，但病人反映未見實質成效，顯示執行與監督機制仍有不足。</span>

<span style='font-weight:bold;background: white'>（二）照護資源配置不均，影響服務公平性</span>
<span style='font-weight:bold;background: white'>問題說明：</span>
<span style='background:white'>「住院整合照護試辦計畫」僅於五北病房提供，包含夜間照護助理（如倒尿、排液處理等），五南病房未納入。</span>
<span style='font-weight:bold;background: white'>影響：</span>
- <span style='background:white'>五南病人與家屬需自行負擔較多照護工作</span>
- <span style='background:white'>護理人員工作負荷增加</span>
- <span style='background:white'>病人感受服務落差（資源不均）</span>
- <span style='background:white'>整體就醫體驗下降</span>
<span style='font-weight:bold;background: white'>補充：</span>
<span style='background:white'>病人曾於五北簽署該計畫，轉至五南後明顯感受差異。</span>

<span style='font-weight:bold;background: white'>（三）夜間人力不足，影響照護效率</span>
<span style='font-weight:bold;background: white'>問題說明：</span>
<span style='background:white'>五南病房夜間護理人力不足，護理人員需同時處理給藥、突發狀況與基礎照護，難以及時回應所有需求。</span>
<span style='font-weight:bold;background: white'>影響：</span>
- <span style='background:white'>夜間照護可能延遲</span>
- <span style='background:white'>護理人員過度負荷</span>
- <span style='background:white'>家屬需介入照護</span>
- <span style='background:white'>潛在醫療安全風險</span>
<span style='font-weight:bold;background: white'>案例補充：</span>
<span style='background:white'>術後需頻繁排液（約每小時一次），夜間負擔尤重。</span>

<span style='font-weight:bold;background: white'>四、病人期望</span>
<span style='font-weight:bold;background: white'>（一）提升清潔品質與感染控制</span>
- <span style='background:white'>落實每日清掃與拖地作業</span>
- <span style='background:white'>禁止病房與廁所共用清潔工具</span>
- <span style='background:white'>建立害蟲防治機制</span>
- <span style='background:white'>確保清潔流程符合感染控制標準</span>

<span style='font-weight:bold;background: white'>（二）擴大照護助理服務範圍</span>
- <span style='background:white'>將整合照護或類似服務擴展至武南病房</span>
- <span style='background:white'>提供夜間基礎照護支援</span>
- <span style='background:white'>降低家屬與護理人員負擔</span>
- <span style='background:white'>提升服務公平性</span>

<span style='font-weight:bold;background: white'>（三）改善夜間人力配置</span>
- <span style='background:white'>優化夜班排班機制</span>
- <span style='background:white'>增加支援人力</span>
- <span style='background:white'>強化照護流程分工</span>
- <span style='background:white'>提升夜間照護效率</span>

<span style='font-weight:bold;background: white'>五、其他重點整理</span>
- <span style='background:white'>病人已轉院，仍願提供建設性回饋</span>
- <span style='background:white'>對護理人員個人表現給予高度肯定</span>
- <span style='background:white'>問題核心集中於制度與資源配置</span>
- <span style='background:white'>顯示目前改善措施存在「執行落差」</span>
