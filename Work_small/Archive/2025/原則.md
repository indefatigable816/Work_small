<span style='color:black'>**一、核心總原則（不可妥協）**</span>
- «span style='color:black'»**保持清醒與簡潔  **
  以哲學思考看世界，誠實面對人性與自身弱點。  
  克制慾望，過原則導向的簡單生活，讓人生變得清晰而有力。«/span»
- «span style='color:black'»**No shortcut. Build systems.  **
  不投機，不幻想捷徑；用紀律與系統累積實力。«/span»
- «span style='color:black'»**行動比情緒重要  **
  人生意義不是想出來的，是做出來的。«/span»
- <span style='color:black'>**未照計畫完成，通常不是能力問題，而是成癮或逃避。**</span>

<span style='color:black'>**二、對自己的紀律（Self-Discipline）**</span>
<span style='color:black'>**1. 行動與成長**</span>
- <span style='color:black'>遠大目標必須拆成**具體、可行動的小步驟**。</span>
- <span style='color:black'>想到就做，立刻行動。</span>
- <span style='color:black'>**逼自己是一種能量來源**：越行動，越有電。</span>
- <span style='color:black'>每天做的事，必須與長期方向一致。</span>
- <span style='color:black'>專注於自我精進，而非短期回報或金錢。</span>
<span style='color:black'>**2. 時間與專注**</span>
- <span style='color:black'>明確區分**重要 / 不重要**。</span>
- <span style='color:black'>時間只花在重要的事上。</span>
- <span style='color:black'>被約時間時，清楚講明：**時間、地點、目的**。</span>
- <span style='color:black'>行動前或行動後，盡快釐清「這件事的終點是什麼」（以終為始）。</span>
<span style='color:black'>**3. 生活簡化**</span>
- <span style='color:black'>減少不必要的慾望與衝動。</span>
- <span style='color:black'>生活越簡單，行動力越強。</span>
- <span style='color:black'>保持環境、身心乾淨有序。</span>

<span style='color:black'>**三、身體與能量管理（Body = Power）**</span>
- <span style='color:black'>吃飽、喝水、營養均衡。</span>
- <span style='color:black'>作息規律，維持穩定節奏。</span>
- <span style='color:black'>身體是所有行動力的根本，不能透支。</span>
- <span style='color:black'>簡單、自律，本身就是力量。</span>

<span style='color:black'>**四、人際原則（Interpersonal Discipline）**</span>
<span style='color:black'>**1. 成為讓人安心的存在**</span>
- <span style='color:black'>微笑、積極、好奇。</span>
- <span style='color:black'>管理自己的形象：**有你在場，別人會放鬆，知道你站在他們那邊**。</span>
- <span style='color:black'>偶爾帶大家一起安靜思考。</span>
<span style='color:black'>**2. 溝通與互動**</span>
- <span style='color:black'>回覆他人：正向、簡潔。</span>
- <span style='color:black'>別人說話時：多聽、多停頓、不打斷。</span>
- <span style='color:black'>做錯事，**立刻道歉**，不辯解。</span>
- <span style='color:black'>表達不是「說」，而是**對聽眾說**：視覺化、貼近對方在意的事。</span>
- <span style='color:black'>準備一兩個分享見聞的故事，搭配肢體語言。</span>
<span style='color:black'>**3. 理解他人**</span>
- <span style='color:black'>留意並記住他人真正重視的事。</span>
- <span style='color:black'>表達前先為對方設想。</span>
- <span style='color:black'>語氣與行動，往往就是他人做決定的依據。</span>

<span style='color:black'>**五、團隊與使命感（Team & Mission）**</span>
- <span style='color:black'>**先把事情做好，不要先想賺錢**。</span>
- <span style='color:black'>以使命感出發，專注做對的事，生活自然會好。</span>
- <span style='color:black'>多才多藝對個人未必重要，**但對團隊是如虎添翼**。</span>
- <span style='color:black'>團隊中：</span>
  - <span style='color:black'>目標要說清楚。</span>
  - <span style='color:black'>方向要說清楚。</span>
  - <span style='color:black'>主動踏出第一步，發出邀請。</span>
- <span style='color:black'>成為讓團隊更穩、更強的人。</span>

<span style='color:black'>**六、守護重要的人（Responsibility）**</span>
- <span style='color:black'>對家人、伴侶、摯友保持愛與包容。</span>
- <span style='color:black'>嚴以律己，寬以待人。</span>
- <span style='color:black'>讓身邊的人，因你的存在而感到溫暖與安心。</span>
- <span style='color:black'>**成為能夠領導自己人生的人**，而不是被情緒與慾望牽著走的人。</span>

<span style='color:black'>**七、人生態度（Daily Orientation）**</span>
- <span style='color:black'>勇於探索與創造：運動、閱讀、音樂、寫作。</span>
- <span style='color:black'>保持生命的流動感與靈動感。</span>
- <span style='color:black'>在理性與感性、事業與生活、追夢與陪伴之間，維持動態平衡。</span>
- <span style='color:black'>**銳利前行，也柔軟以待**。</span>
- <span style='color:black'>希望世界是什麼規則，就親自去實踐它。</span>

<span style='color:black'>**每日一句錨點（建議每天讀）**</span>
<span style='color:black'>我選擇清醒、簡單、自律。</span>
<span style='color:black'>我用行動建立系統，用紀律創造自由。</span>
<span style='color:black'>我把事情做好，守護重要的人，活出值得尊敬的人生。</span>
