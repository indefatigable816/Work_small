<span style='color:black'>25夏．美國</span>
<span style='color:black'>**從書本裡的美國文化，到真的踏上那片土地。這趟交換旅程，實驗室的挑戰讓我體會到研究的艱辛，也提醒自己要找到能與人生方向複利的主題去深耕。**</span>
<span style='color:black'>**在紐約，我盡可能融入當地：打球、聊天、聽音樂會、走進博物館，也嘗試各國料理。身邊的人來自不同國度，每一次交流都是新的視角。這讓我更清楚，自己在世界裡渺小如一粒沙，但依舊能選擇專注與成長。**</span>
<span style='color:black'>**離開前，思鄉之情前所未有地強烈。但見識越多，就越想繼續變強。**</span>
<span style='color:black'>成長過程中，從國小到國高中，我們學習英文時就無形中在接收許多美國文化。其範圍含括生活中的大小事，小至流行穿著，大至重要歷史。好奇心總是驅使我們問：「那究竟是一個怎樣的地方？」美國夢，那是大家曾經的夢想。</span>
<span style='color:black'>常聽說誰誰誰家孩子送去美國讀書，常聽說紐約又修改了什麼法案。每次聽到，我心想都只是與我何干？或許因為美國是大家口中的世界老大，明明生活在台灣，卻總感覺美國發生的事大家還是很在乎的。</span>
<span style='color:black'>因緣際會，我獲得了一次取美國暑期交換的機會。非常感恩，因而非常認真地在當地的實驗室中學習，同時最大化的利用這次的機會。</span>
<span style='color:black'>實驗室分派給我具體的專案，我和其他一同前往的台灣學生們共同努力完成任務，並在過程中解決問題。我理解到進行基礎研究是困難的，要在努力翻閱前人相關研究，找出自身要如何在更進一步解決問題的同時，努力解決眼下使用工具遇到的問題。由於做研究要投入大量精力與時間，所以我認為將來我在做研究時將挑選更能跟我人生發展符合，並能造成複利的主題。接下來半年我想研究各科別未來的職業狀況，並閱讀我最有興趣科別的文章，半年後希望可以到有興趣的實驗室。</span>
<span style='color:black'>（在見識到實驗室大家的表達能力後，我發現自己的不足。一直以來我都是不斷輸入而卻乏輸出，因此我想要在大量閱讀書籍的同時，透過寫作來增強我的表達能力。）</span>
<span style='color:black'>在美國，我也盡可能地去體驗當地人的生活模式和節奏。嘗試了許多自己不喜歡的披薩、潛艇堡、炸物，每周都去中央公園球場和不同人交流球技並閒聊生活，去林肯中心、布萊恩特公園、百老匯參加各種音樂盛宴，像當地音樂家一樣去琴房練琴、去地鐵站表演，漫步在紐約的各個街道上，韓國城、中國城、黑人區、時代廣場...，動物園、博物館、游泳池、海岸、夕陽...，逛過各種畫廊，跟各種路人、店員聊天，嘗試各國的料理，地鐵上有各種膚色的人，租的房間裡也有各種國家的人，都是一方的大人物。那是一種生活方式，一種我不熟悉的生活方式。實驗室中許多華人夥伴，有的離鄉背井到此為求得更好的發展，有的從小便闖蕩在世界各地，我的出身和原本的思維讓我成了個台灣小小的醫學生。人們總人們總說：「在宇宙中，我們就是一粒沙」，但我總覺得，光在地球上我就已經是一粒沙了呢？</span>
<span style='color:black'>（因此我應該要認清楚自己的影響圈，目光遠大是好事，但凡是大事都要從小事做起，在生活中維持對慾望的控制，對目標的專一，足夠專注才能過有趣的人生）</span>
<span style='color:black'>這趟旅程最有趣的體驗是去一位因緣際會認識的朋友家玩。之前在台灣跟他一起在高雄美術館（我第一次去）和駁二一帶遊玩過，他性格較內向，比較少主動找我說話。他是樂團長笛首席，曾在少年就獲音樂名校肯定，在林肯中心出過獨奏會。現在在美國讀醫，他在圖書館會看建築的書和科技史的書。在跟他相處的過程中，我發現有趣的是一個話題可以拿出來聊三四次，每次越聊越深，是一種我沒有體驗過的聊天模式。在美國，先是約他一起逛大都會，他人很好說話也挺可愛，所以跟她相處算是相當愉快的。</span>
<span style='color:black'>（我對這種很好的人總是存著戒備，因為我認為人的本性暴露的時候總是非常可怕，不能因為一個人在某些方面好，就去相信他在日後不會傷害你。要劃清界線保持清醒，不要投射美好到他人身上，要相信只有自己可以在生活中創造出美好，世界不會因為一個人變好）</span>
<span style='color:black'>後來，我去他家玩，他跟他弟感情挺好的，所以我就沒有說太多話，默默觀察他怎麼寵弟弟。他開車載我到長島上的各處遊玩，高速公路邊是海灘、是無盡的草原，草原上有牛群、小鹿、野鴨。我們去了花園，散步在玫瑰花中，偶爾聊著音樂，偶爾各自靜賞風景，我向她傾訴在紐約遇上的各種新鮮事，他含蓄的表達他對台灣的喜愛，細緻的水滴在炎熱的夏日中澆灑，心情輕快涼爽。回想起來，和美女逛花園真是過分浪漫了。姊弟倆廚藝很不錯，雖然那幾天不知道是因為他們的食物還是HMart的食物而拉肚子，但被照顧得很好很開心。</span>
<span style='color:black'>賦歸，果然在異地因為文化背景不同，要建立深度連結不是那麼的容易，思鄉之情從未如此強烈。見識了這麼多之後，雖然很中二，但我想要繼續變強。</span>
<span style='color:black'>25夏．交大網球</span>
<span style='color:black'>時間很快來到暑假的最後一周，在球隊參加了為期7天的網球訓練。週三我們移至交大，每天規律的訓練打球6小時，期間專注力全然地放在球上，即便汗水已盡失的衣褲和場地，四肢有些疲累，肌肉異常緊繃，手腳有些破皮，那些都阻止不了我用力打出每一顆球的熱血。</span>
<span style='color:black'>很感謝有語倢來陪我這麼多天，在我累的時候照顧我，支援我做很多事情，在情人節買了7朵白色薔薇給我們，陪我去巨城吃飯，陪我睡覺，最喜歡有浪漫，也有兩個人各自在為成長而努力。</span>
<span style='color:black'>這次在交大也認識了許多電機系的朋友，能跟很厲害的人們一起練球實在很幸福，這周練球把我去美國吃肥的地方都變成肌肉了，外表看起來很瘦，BMI卻達到24了，沒想到我也會有過重的這一天。</span>
<span style='color:black'>25夏．花蓮</span>
<span style='color:black'>周一忽然想起來跟奶奶的約，周三便訂了車票和住宿，週五實驗課下課就飛快前去搭乘火車，是第一次和語倢搭火車。第一晚語倢很累，秒睡，我跟蚊子搏鬥。週六一早到阿公家就開始忙普度的事，午餐嚐了奶奶熟悉的手藝，下午去旁邊遠百懷舊、買鞋、拍貼，傍晚阿公讓我開車去壽豐的Starbuck享受下午茶，日落月浮，滿月自深藍海面升起，灑下片片金鱗，輕輕鋪在海面上。晚餐在日式眷村將軍府，享受咖哩，和爺奶的溫情。</span>
<span style='color:black'>晚上回到飯店終於直面小語倢，心生陌生感，性味全無，隱隱有些恐懼。語倢面臨陰道痙攣，而我面臨心因性勃起障礙，是不愉快的第一次，我難過的在被窩中哭泣，語倢輕聲安慰我。隔天中午，阿公帶我們去吃六里屯牛肉麵，他們家的魚蛋超香，讚不絕口。</span>
<span style='color:black'>這次出遊很放鬆，讓我身心重新整頓，回歸校園認真讀書工作。</span>
<span style='color:black'>本學期目標：盡力全A+、找到復健科/骨科相關研究室、藥物開發專案完成、課外閱讀拓展知識。感情上自我要求高一點，我覺得回到原本剛交往的那種距離也不錯，可以把相處所需系統化。家人就是懷著感恩之情和明確界線處理。</span>
