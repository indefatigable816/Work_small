PM

- <span style='color:black'>情緒形成公式</span>
  - <span style='color:black'>type1: 保命的情緒: 面對劍齒虎恐懼</span>
  - <span style='color:black'>type2: **詮釋(某事件/想法)+認同+重複**</span>
- <span style='color:black'>我不夠好</span>
  - <span style='color:black'>事實是，你對大部分的事情其實都做得不錯，雖然缺乏經驗、興趣和天賦，都可以用來解釋為何你在一些領域未能達成自己的期望，但這于你不夠好無關。</span>
  - <span style='color:black'>此感覺與自尊心低落有關</span>
    - <span style='color:black'>對過去經歷的負面解讀</span>
  - <span style='color:black'>把事件拿出來重新分析:**詮釋(某事件/想法)+認同+重複**</span>
  - <span style='color:black'>寫勝利日誌</span>
  - <span style='color:black'>正面日記</span>
- <span style='color:black'>自尊六大支柱</span>
  - <span style='color:black'>有意識的生活</span>
  - <span style='color:black'>自我接納: 珍視尊重自己</span>
  - <span style='color:black'>自我負責: 沒有人會來救你，只有你能改變自己的人生</span>
  - <span style='color:black'>自信果敢: 尊重自己的渴望需求，並以適當形式表達</span>
  - <span style='color:black'>有目標的生活: 運用能力達成目標</span>
  - <span style='color:black'>正直並依價值觀行事(正義)</span>
- <span style='color:black'>防衛心</span>
  - <span style='color:black'>當強烈認同的信念受到攻擊，會出現情緒反應</span>
- <span style='color:black'>在意別人對你的想法</span>
  - <span style='color:black'>理解別人並不在乎你</span>
  - <span style='color:black'>理解你並不在乎別人</span>
  - <span style='color:black'>放下自我形象</span>
    - <span style='color:black'>人們永遠會根據他們的價值觀與信念詮釋你的言行</span>
- <span style='color:black'>憂鬱</span>
  - <span style='color:black'>憂鬱時，更多的思考不是解決之道</span>
  - <span style='color:black'>憂鬱是一個徵兆，表示你需要遠離頭腦的思想，放下對過去未來的擔憂，當前情況的詮釋，重新連結到當下</span>
