PM
<span style='color:black'>基本假設</span>
- <span style='color:black'>人們根據自己的期望收益和成本選擇</span>
- <span style='color:black'>遊戲規則：私有產權、法治社會</span>
<span style='color:black'>效率</span>
- <span style='color:black'>值不值得：問的是效率</span>
- <span style='color:black'>價值觀決定我們對效率的看法</span>
- <span style='color:black'>很多時候不是效率問題而是**權力在誰**</span>
- <span style='color:black'>專業化追求比較利益</span>
- <span style='color:black'>交易成本：安排交易的成本，中間人創造資訊降低交易成本</span>
<span style='color:black'>需求</span>
- <span style='color:black'>選擇依賴於目前處境：邊際價值</span>
- <span style='color:black'>需求曲線：需求量</span>
- <span style='color:black'>需求</span>
  - <span style='color:black'>正常才</span>
  - <span style='color:black'>劣等才</span>
  - <span style='color:black'>替代</span>
  - <span style='color:black'>互補</span>
  - <span style='color:black'>預期</span>
  - <span style='color:black'>彈性</span>
    - <span style='color:black'>時間</span>
    - <span style='color:black'>替代品</span>
    - <span style='color:black'>佔預算比例</span>
<span style='color:black'>供給</span>
- <span style='color:black'>成本與行為、選擇有關</span>
- <span style='color:black'>供給曲線、彈性</span>
- <span style='color:black'>對誰的成本、做啥的成本？</span>
<span style='color:black'>利與虧</span>
- <span style='color:black'>合約降低不確定性</span>
- <span style='color:black'>利潤／虧損是不確定性的結果是剩餘物</span>
- <span style='color:black'>企業家／老闆有剩餘請求權</span>
- <span style='color:black'>企業家精神</span>
  - <span style='color:black'>套利：低買高賣</span>
  - <span style='color:black'>發現市場的錯誤：買賣方因資訊不足犯了什麼錯</span>
  - <span style='color:black'>創新</span>
  - <span style='color:black'>模仿</span>
- <span style='color:black'>市場創造資訊</span>
<span style='color:black'>詢價定價</span>
- <span style='color:black'>差別訂價讓淨收益最大化，使邊際收益等於零</span>
  - <span style='color:black'>Lunch Special</span>
- <span style='color:black'>使用差別訂價的交易成本</span>
  - <span style='color:black'>區分不同需求彈性購買者</span>
  - <span style='color:black'>阻止低價購買者賣給高價購買者</span>
  - <span style='color:black'>控制不滿情緒</span>
- <span style='color:black'>若不是因為交易成本會有更多差別訂價</span>
<span style='color:black'>競爭</span>
- <span style='color:black'>完全競爭市場</span>
  - <span style='color:black'>買賣方人數多，資訊完全流通，大家都是價格接受者</span>
  - <span style='color:black'>賣方提供同質性商品</span>
  - <span style='color:black'>資源配置的調動不需成本</span>
- <span style='color:black'>企業家透過法律控制競爭對手量，進入行業的阻力、專利</span>
- <span style='color:black'>只有不確定性存在時，才會有利與虧</span>
  - <span style='color:black'>專利：看似無風險的利潤但</span>
    - <span style='color:black'>需求不確定</span>
    - <span style='color:black'>政策可能改變</span>
    - <span style='color:black'>成本可能上升</span>
    - <span style='color:black'>技術可能被淘汰</span>
    - <span style='color:black'>也可將專利看做過去研發承擔不確定性的報酬</span>
- <span style='color:black'>政府是否管制以避免掠奪式定價?</span>
  - <span style='color:black'>在競爭不能控制市場力量的區域，要</span>
  - <span style='color:black'>限制/支持競爭?</span>
  - <span style='color:black'>支持競爭更有優勢</span>
    - <span style='color:black'>Antitrust: 避免意圖壟斷的行為造成公共福利之威脅</span>
      - <span style='color:black'>限制購併未必限制競爭，也未必能為消費者帶來福祉</span>
      - <span style='color:black'>抓小蝦而放大魚</span>
      - <span style='color:black'>無充分平衡的評估</span>
<span style='color:black'>外部性</span>
- <span style='color:black'>決策者考慮之外的利虧</span>
- <span style='color:black'>外部性內化: 將外部因素納入考慮(前提:體諒的美德)</span>
- <span style='color:black'>透過協商negotiate最小化負外部性</span>
- <span style='color:black'>裁定adjudicate: 判定時擁有甚麼權力</span>
- <span style='color:black'>立法解決外部性:沒有誘因</span>
  - <span style='color:black'>若限制排放量，則會導致大工廠，每單位汙染物減排成本最大的工廠，吸收最多成本，不公平</span>
  - <span style='color:black'>若收排放稅，會使排放成本高於稅收的工廠繼續排放，但徵稅引起民怨</span>
  - <span style='color:black'>只好用汙染許可證</span>
  - <span style='color:black'>汙染權: 若今天環保署將成本全部施加在排放量最大的工廠上，則他會花錢去讓減少排上成本較低的工廠淨零排放。比較利益最大化</span>
<span style='color:black'>市場與政府</span>
- <span style='color:black'>政府: 藉由強制來降低交易成本的制度安排</span>
- <span style='color:black'>覺得自己一票的貢獻很小，蒐集資訊的意願會降低，因為正外部性促使我們採取搭便車行為，而民意代表可能更願意獲得充分資訊再做決定</span>
- <span style='color:black'>政治制度設計的一大重點是: 如何讓公共利益與議員利益達到一致</span>
  - <span style='color:black'>例如連任對議員是有利的，為此他會採取短期內能得到支持但長期卻是虧損的政策</span>
- <span style='color:black'>民主政治過程的邏輯就是:</span>
  - <span style='color:black'>將資訊集中於組織良好、資訊完備、獲利最多的少數人手上</span>
  - <span style='color:black'>將成本分散到沒有組織、資訊缺乏、得不到什麼利益的人身上</span>
- <span style='color:black'>政府政策往往受到特殊利益團體支配，因為人們會根據自己獨特的利益定義公共利益，所以政府寧願對多數人帶來小小的傷害，也不願造成少數人極度不高興</span>
- <span style='color:black'>許多公共利益的困難在於囚徒困境，透過合約法律等約束社會就會都選擇總剩餘最大的決定，然而擬定這些合約的交易成本巨大。我們用微笑、皺眉、風俗習慣、合約保證金、憲法來克服成本保障合作。</span>
<span style='color:black'>GDP</span>
- <span style='color:black'>一國家一年內所生產最終產品和服務的總市值</span>
- <span style='color:black'>未出售物品由原廠商自行購買(存貨投資)，隔年GDP再依實際受出價做調整</span>
- <span style='color:black'>失業率: 失業人口(沒有就業但正積極尋求就業者U3)/勞動力人口</span>
- <span style='color:black'>實質GDP: 固定購買力的貨幣價衡量；名目GDP: 當前貨幣價格下的GDP</span>
<span style='color:black'>貨幣</span>
- <span style='color:black'>能使某種東西變成貨幣的，是他事實上作為交易媒介被人們接受並使用</span>
- <span style='color:black'>銀行持有客戶存款的法定準備金比率，其他超額準備可以借出去獲取利潤，形成存款擴張乘數，準備金比率之倒數</span>
- <span style='color:black'>準備金不足會像聯準會貸款，其利率稱為貼現率discount rate，聯準會透過公開市場操作可以控制貨幣供給</span>
