PM

- <span style='color:black'>三種思考</span>
  - <span style='color:black'>邏輯思考：明確化各項要素的關係，可爬梳資訊但未必正確</span>
  - <span style='color:black'>水平思考：多觀點思考，打破框架</span>
  - <span style='color:black'>批判性思考：用邏輯批判情報資訊</span>
    - <span style='color:black'>1. 質疑框架</span>
    - <span style='color:black'>2. 質疑假設</span>
- <span style='color:black'>MECE: Mutually exclusive, collectively exhaustive</span>
- <span style='color:black'>unconcious bias:</span>
  - <span style='color:black'>authority bias</span>
  - <span style='color:black'>normalcy bias: 應該不會那麼嚴重吧</span>
  - <span style='color:black'>confirmation bias</span>
  - <span style='color:black'>halo effect: 受突出特質影響</span>
  - <span style='color:black'>primacy effect</span>
  - <span style='color:black'>recency effect</span>
  - <span style='color:black'>just-world hypothesis: (人們難以承認世界本不公平，容易怪罪受害者)</span>
- <span style='color:black'>媒體一直播報一方向的資訊時，可能在掩蓋稀釋另一方向的資訊</span>
