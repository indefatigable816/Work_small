- <span style='color:black'>次對角線上是非黑即白的世界</span>
- <span style='color:black'>只停留在一個象限會形成意識形態，相同意識形態者形成族群，區分我們和你們</span>
- <span style='color:black'>政治人物鎖定特定族群稱為**身分政治**</span>
  - <span style='color:black'>法西斯主義者：去人性化少數族群加以迫害</span>
- <span style='color:black'>觀眾獲得的資訊越不充分就越容易受影響，知識空白之處就用宣傳填滿</span>
- <span style='color:black'>判別宣傳的關鍵步驟：誰最大獲益者</span>
- ![image2](../../../../resources/061c7092c3b542d7a8f3cbcbdb063d51.png)
- <span style='color:black'>“It ain't ignorance causes so much trouble; it's folks knowing so much that ain't so.”—— Josh Billings, Everybody's Friend, or Josh Billing's Encyclopedia and Proverbial Philosophy of Wit and Humor (1874)</span>
- <span style='color:black'>宣傳是一門藝術</span>
  - <span style='color:black'>關鍵</span>
    - <span style='color:black'>簡單二元化的故事</span>
    - <span style='color:black'>隱性偏誤</span>
    - <span style='color:black'>激發負面情緒</span>
    - <span style='color:black'>促使民眾區分你們我們</span>
    - <span style='color:black'>目的：讓權力階級得利</span>
  - <span style='color:black'>技巧</span>
    - <span style='color:black'>引發懷疑：規避責任，讓有權勢者(我們會較相信他們)出來質疑</span>
    - <span style='color:black'>Framing: 讓同樣的事聽起來比較好聽</span>
      - <span style='color:black'>戰爭部、國防部</span>
      - <span style='color:black'>虐待、審訊技巧</span>
      - <span style='color:black'>歐威爾式語言：用相反的好詞去稱呼一件壞事</span>
    - <span style='color:black'>人生攻擊、誹謗、破壞名聲</span>
      - <span style='color:black'>Feminazi</span>
      - <span style='color:black'>自由主義者</span>
        - <span style='color:black'>常被黨媒連結到對傳統社會進行徹底變革</span>
        - <span style='color:black'>其原意單純指進步與改革</span>
        - <span style='color:black'>強調個體自由與政府保障個體權利</span>
    - <span style='color:black'>轉移焦點</span>
      - <span style='color:black'>Whataboutism</span>
        - <span style='color:black'>故意將話題帶離指控者有效論點</span>
        - <span style='color:black'>用毫不相關案例作不當對等比喻</span>
        - <span style='color:black'>A 說：「你上次報告的數據有錯。」</span>
        - <span style='color:black'>B 回答：「你上次也不是弄錯過報告嗎？」</span>
