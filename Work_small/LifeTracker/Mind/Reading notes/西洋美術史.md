  - <span style='color:black'>男子選美大會（各年齡層）</span>
  - <span style='color:black'>從小接受軍事訓練 透過服兵役獲得選舉權</span>
  - <span style='color:black'>plato的本名是aristokles意指肩膀寬</span>

  - <span style='color:black'>Kouros少年像;Kore少女像</span>
  - <span style='color:black'>對置姿勢contrapposto</span>
- <span style='color:black'>古典時期</span>
  - <span style='color:black'>雅典與斯巴達敵對，捲入伯羅奔尼撒戰爭</span>
    - <span style='color:black'>伯羅奔尼撒戰爭（前431–404）為雅典領導的提洛同盟與斯巴達領導的伯羅奔尼撒同盟之間的長期戰爭。主因為雅典的擴張與斯巴達的安全恐懼。關鍵事件有雅典的瘟疫與佩里克利逝世、西西里遠征（雅典慘敗）、波斯資助斯巴達造艦與封鎖。最後雅典投降，海上霸權瓦解，希臘城邦整體衰弱，為馬其頓後來崛起創造條件。</span>
  - <span style='color:black'>恐怖政治促成優美風格eg.Praxiteles</span>
  - <span style='color:black'>缺乏表情：希臘人認為控制情緒為善</span>
- <span style='color:black'>希臘化時期</span>
  - <span style='color:black'>亞歷山大三世遠征波斯，建立大帝國，散播希臘文化</span>
  - <span style='color:black'>融合希臘以外價值觀：理想－＞寫實；敬神物－＞美術品</span>
- <span style='color:black'>原作幾乎都佚失，存至今多為羅馬人翻刻</span>
- <span style='color:black'>政治上羅馬支配希臘，文化上希臘征服羅馬</span>
- <span style='color:black'>奧林匹克</span>
  - <span style='color:black'>祭神儀式，使城邦需停戰</span>
  - <span style='color:black'>原限希臘人參加，直到羅馬統治希臘，但羅馬人不喜歡裸體競技</span>
  - <span style='color:black'>冠軍獲紅絲帶，宙斯神殿附近的橄欖枝，象徵和平</span>
  - <span style='color:black'>AD 349迪奧多西定基督教為國教，獻給宙斯的奧林匹克閉幕</span>
<span style='color:#434343'>羅馬美術</span>
- <span style='color:black'>流源：希臘和’’Etruria’’</span>
  - <span style='color:black'>全盛於6,7century BC</span>
  - <span style='color:black'>Necropolis: 豪華墓地</span>
  - <span style='color:black'>Etruria特有半圓形拱門</span>
- <span style='color:black'>共和，奧古斯都後走向獨裁，五賢帝時代版圖最大</span>
- <span style='color:black'>Pax Romana 地中海和平經濟繁榮</span>
- <span style='color:black'>奢華</span>
- <span style='color:black'>祭祖:蠟頭像/胸像，不同於希臘人愛全身像</span>
- <span style='color:black'>希臘人視皺紋為醜惡；羅馬視為敬意</span>
- <span style='color:black'>宏偉建築</span>
  - <span style='color:black'>圓形競技場</span>
    - <span style='color:black'>為操縱人心而建:格鬥士、處刑</span>
    - <span style='color:black'>古羅馬最大建築</span>
  - <span style='color:black'>公共浴場</span>
    - <span style='color:black'>含音樂戲劇健身功能</span>
  - <span style='color:black'>功臣:混凝土:水泥+砂礫</span>
  - <span style='color:black'>凱旋門</span>
    - <span style='color:black'>紀念軍事勝利</span>
    - <span style='color:black'>拿破崙自喻羅馬皇帝建雄獅凱旋門</span>
  - <span style='color:black'>萬神殿</span>
    - <span style='color:black'>建築風格影響文藝復興</span>
- <span style='color:black'>羅馬帝國衰退伴隨基督美術萌芽</span>
  - <span style='color:black'>因異族反覆入侵改四帝共治，<u>四帝共治像</u>代表了寫實－＞象徵</span>
  - <span style='color:black'>表現內在的世界</span>

  - <span style='color:black'>君士坦丁遷都拜占庭</span>
<span style='color:#434343'>宗教美術、仿羅馬式美術</span>
- <span style='color:black'>中世紀的歐洲由基督教支配</span>
- <span style='color:black'>五世紀的五大教會：羅馬、君士坦丁堡、耶路撒冷、安提阿、亞歷山大－＞五大牧首區</span>
- <span style='color:black'>基督教背景</span>
  - <span style='color:black'>原猶太教徒耶穌(拿撒勒人)發起宗教改革運動</span>
  - <span style='color:black'>相信救世主/基督=耶穌</span>
  - <span style='color:black'>舊約聖經=猶太教的希伯來聖經</span>
- <span style='color:black'>基督教普及的核心：羅馬教會、法蘭克王國</span>
- <span style='color:black'>法蘭克王國(日耳曼人)：AD 5~9 century支配西歐</span>
  - <span style='color:black'>盛世：Merovingian dynasty (beginning), Carolingian dynasty (ending)</span>
- <span style='color:black'>基督美術起源</span>
  - <span style='color:black'>Merovingian dynasty第一任國王Clovis I 受妻子影響受洗為基督徒。羅馬教會更易在其境內傳教</span>
  - <span style='color:black'>舊約說禁看神臉，摩西十戒說避免偶像崇拜，但為了向Alps北部的人傳教，將聖經繪製成畫</span>
- <span style='color:black'>Merovingain dyn.持續2世紀，實權到宰相鐵鎚查理手中(Charles Martel)，其子矮子丕平廢黜最後一位國王，進入加洛琳王朝</span>
- <span style='color:black'>羅馬教會</span>
  - <span style='color:black'>矮子丕平為了證明正統性，接受羅馬教宗的塗油加冕儀式(Saint-Denis)</span>
  - <span style='color:black'>其子Charlemangne查里曼隨後接受加冕成為西羅馬皇帝</span>
  - <span style='color:black'>皇權與羅馬教會因而變緊密</span>
- <span style='color:black'>After Charlemangne died, 法蘭克帝國分裂，歐洲北部混亂</span>
- <span style='color:black'>修道士便遠離城市建修道院、教堂(仿羅馬式藝術11.5-13century)</span>
- <span style='color:black'>10世紀後石造建築增加，更符合神之家的形象</span>
- <span style='color:black'>區分仿羅馬與歌德:半圓與尖頂</span>
- <span style='color:black'>摩西十戒說：禁止偶像崇拜，因此早期教堂裝飾多為壁畫，但仿羅馬時期後開始有雕像，為寫實主義，為了打動不識字的人</span>
- <span style='color:black'>民眾感受到強烈的末日感，出發朝聖之旅(crusade)，修道院形成交通要道，發展出住宿、市集，帶回聖遺物，人們相信那可以幫助它們實現願望，當時沒有一件教堂不收藏聖遺物(帶來知名度和市集收益)</span>
- <span style='color:black'>朝聖帶來都市發展</span>
- <span style='color:black'>基督宗教或承認AD313以前的基督美術</span>
  - <span style='color:black'>葬禮美術：墓穴壁畫石棺</span>
  - <span style='color:black'>基督－＞好牧人－＞借用羅馬葬禮的牧羊人風景</span>
  - <span style='color:black'>秘密符號(魚): 耶穌基督、神之子、救世主的字首合起來是魚</span>
<span style='color:#434343'>哥德式美術</span>
- <span style='color:black'>法蘭克王國(路易五世)－＞法蘭西王國(卡佩王朝)－＞．．．－＞法國</span>
- <span style='color:black'>Suger作為院長改建Saint-Denis－＞哥德樣式(Goth野蠻人，文藝復興時期義大利人的輕蔑稱呼)</span>
  - <span style='color:black'>政治意圖：透過新的建築樣式在法國各地宣示法蘭西王國的權勢</span>
  - <span style='color:black'>因人口集中形成都市，為了教堂能容納更多人</span>
